[PyGMT](https://www.pygmt.org/) is a Python interface to the Generic Mapping Tools (GMT) — the standard tool for publication-quality geoscience maps.

- [PyGMT documentation](https://www.pygmt.org/latest/)
- [PyGMT gallery](https://www.pygmt.org/latest/gallery/) — copy-paste examples for common map types
- [GMT forum](https://forum.generic-mapping-tools.org/)

**Installation:**

```bash
conda install -c conda-forge pygmt
```
