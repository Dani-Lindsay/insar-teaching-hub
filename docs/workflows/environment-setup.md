## Quickest start — OpenSARLab

[OpenSARLab](https://opensarlab-docs.asf.alaska.edu/) is a managed JupyterHub environment provided by ASF with MintPy and other InSAR tools pre-installed. No local setup required — log in with your NASA Earthdata account and run notebooks in the browser. This is the recommended starting point if you just want to run MintPy without fighting a conda environment first.
